4. 예외로 내가 틀렸던 문제
- 1부터 10까지의 숫자를 반복문을 돌려 호출 후 3의 배수는 숫자가 아닌 "짝"으로 호출하게 하시오.

원래 내가 썼던 코드는
```js
for(let i = 1; i <= 10; i++){
    if(i % 3 = 0) {
        console.log(짝);
    }
    console.log(i);
}

수정 후 정답은

for(let i = 1; i <= 10; i++) {
    if(i % 3 === 0) {
        console.log("짝");
    } else {
        console.log(i);
    }
}
```
